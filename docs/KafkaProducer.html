Name Type Attributes Description
name String The name of current instance.
kafkaHost String <optional>
 The host of the broker of kafka, when both of `zookeeperHost` and `kafkaHost` passed, the `kafkaHost` has higher priority.
topic String <optional>
 The topic where you save data in it.
delayInterval Number <optional>
 When pass this parameter, messages will publish to kafka every `option.delayInterval` ms, otherwise messages will publish to kafka at once.
